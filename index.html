🔑 Connect your Groq key

SayDay AI uses your Groq API key to power real responses. Your key is stored only in this browser session and never sent to any server other than OpenAI.

⚠️ Serving requirement: Open this file via a local server (e.g. VS Code Live Server, python -m http.server) — not by double-clicking it. Browsers block API calls from file:// pages.

Get a key at console.groq.com/keys. Without a key, SayDay runs in offline demo mode.
